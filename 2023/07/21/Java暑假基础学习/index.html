Java 暑假基础学习

Java开发注意事项

  1. 执行入口是 main() 方法

    public static void main(String[] args){}

  2. 严格区分大小写

  3. 最后以 ; 结束

  4. 一个源文件只能有一个public类。其他种的个数不限

  5. 一个源文件如果有public类，则文件名必须按该名命名

转义字符

  1. \t 一个制表位，对齐的功能

  2. \n 换行

  3. \\ 一个

  4. \r 一个回车

文档注释

  1. 注释内容可以被 JDK 中工具 javadoc 所解析，生成一套以网页文件形式体现该程序的说明文档

Java 代码规范

  1. tab 缩进

  2. 运算符和 = 两边习惯性加一个空格

  3. 源文件使用 utf-8 编码

  4. 行宽度不要超过 80 字符

变量

加号左右两边都是数值型时，做加法运算，加号左右有一方为字符串时，则做拼接运算

基本数据类型

  1. 数值型

    • 整数型，byte[1]， short[2]，int[4]，long[8]

    • 浮点类型，float[4]，double[8]    

  2. 字符型，char[2]，存放单个字符

  3. 布尔型，boolean[1]，存放 true，false

引用数据类型

  1. 类 class

  2. 接口 interface

  3. 数组 [ ]

整型

  • 注意：声明了long型常量必须后加L

    常见错误：int n1 = 1L;   

    原因：左边是 int 类型，右边是 long 类型，左右不匹配

浮点型

  • float 单精度，4个字节

  • double 双精度，8个字节

  • 存放形式：浮点数=符号位+指数位+尾数位

浮点型常量默认是 double 型，声明 float 型常量后面必须加 "F"

浮点数使用陷阱

在我们对运算结果是小数的进行相等判断时，要小心

不能直接 == ，应该是以两个数的差值的绝对值，在某个精度范围内判断

字符类型（char）

单个字符用char，char 是两个字节（可以存放汉字），多个字符我们用字符串String

  • 字符常量是单引号（' '）括起来的单个字符

  • 转移符号（）

布尔类型

boolean 型只允许取值 true 和 false，无 null，占一个字节，适用于逻辑运算

与C/C++不同，不可以用 0 或非 0 的整数代替 false 或 true

String 字符串

  • 基本数据类型转 String 类型

        将基本类型的值 + " "

逻辑运算符

&& 和 & 基本规则

都是：两个条件条件为 true，结果为 true

区别：

  • && 短路与：如果第一个条件为 false ，第二个条件不会判断，结果为 false，第一个为假，第二个不会去执行

  • & 逻辑与：不管第一个条件是否为false，第二个条件都要判断，效率低，无论对错都会去执行两个条件

|| 和 |基本规则

都是：两个条件只要有一个成立，结果为 true

区别：

  • || 短路或： 如果第一个条件为 true，则第二个条件不会判断（不会执行），结果为 true，效率高

  • 逻辑或：无论第一个条件是否为 true。第二个条件都要判断，效率低

! 取反基本规则

将条件反过来，如果条件成立，结果为 false ，否则为 true

^ 逻辑异或

a^b ：当 a 和 b 不同时，结果为 true，否则为 false                               

标识符的命名规则和规范

规则（必须遵守）

  1. 由26个英文字母大小写，0-9，_或$ 组成

  2. 数字不可以开头

  3. 不可以使用关键字和保留字，但可以包含

  4. 严格区分大小写，但长度无限制

  5. 标识符不能包含空格

命名规范

  1. 包名：全部小写

  2. 类名，接口名：多单词组成时，所有单词的首字母大写

  3. 变量名，方法名：多单词组成时，第一个单词首字母小写，第二个单词开始每个单词首字母大写

  4. 常量名：所有字母都大写，多单词时用下划线连接
